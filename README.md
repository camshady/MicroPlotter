# MicroPlotter
Desktop 2D pen-plotter using servos and Arduino

--INCOMPLETE--
Connect Arduino to PC via USB, Python script reads gcode and sends arduino instructions over serial. 
Arduino moves 2 servos in SCARA pattern to draw plots.
